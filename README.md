# Creative Agency Website

A responsive digital agency website built with HTML, CSS and JavaScript.

## Technologies Used
- HTML5
- CSS3
- JavaScript 
- Owl Carousel 2
- Font Awesome 6
- Google Fonts (Poppins)

## Features
- Page loader with fade effect
- Sticky navbar
- Mobile responsive hamburger menu
- Lazy loading images
- Owl Carousel sliders (Portfolio + Testimonials)
- Contact form with success message
- Smooth scroll

## Sections
- Hero
- Services
- Portfolio
- About
- Testimonials
- Team
- Contact
- Footer

## How to Run
Just open `indexx.html` in any browser. No setup needed.
